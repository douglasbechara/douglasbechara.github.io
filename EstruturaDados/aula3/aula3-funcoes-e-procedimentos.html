Estrutura de Dados: Funções e Procedimentos

Introdução

Funções e procedimentos são blocos de código reutilizáveis que ajudam a modularizar e organizar um programa. Funções retornam valores após sua execução, enquanto procedimentos realizam operações sem retornar um valor específico.

Exemplos de Funções e Procedimentos

Visualg

// Procedimento para exibir uma mensagem
procedimento exibirMensagem()
inicio
    escreva("Olá, bem-vindo à aula de Funções e Procedimentos!")

funcao somar(x: inteiro, y: inteiro): inteiro

fimfuncao

Python

# Procedimento (função sem retorno) para exibir uma mensagem
def exibir_mensagem():
    print("Olá, bem-vindo à aula de Funções e Procedimentos!")

# Função que retorna a soma de dois números
def somar(x, y):
    return x + y  # Retorna a soma de x e y

C

#include <stdio.h>

// Procedimento em C (função void) para exibir uma mensagem
void exibirMensagem() {
    printf("Olá, bem-vindo à aula de Funções e Procedimentos!\\n");
}

// Função que retorna a soma de dois números
int somar(int x, int y) {
    return x + y;  // Retorna a soma de x e y
}

Java

public class FuncoesProcedimentos {

    // Procedimento em Java (método sem retorno) para exibir uma mensagem
    public static void exibirMensagem() {
        System.out.println("Olá, bem-vindo à aula de Funções e Procedimentos!");
    }

    // Função que retorna a soma de dois números
    public static int somar(int x, int y) {
        return x + y;  // Retorna a soma de x e y
    }

    public static void main(String[] args) {
        exibirMensagem();  // Chama o procedimento
        int resultado = somar(10, 20);  // Chama a função e armazena o resultado
        System.out.println("A soma é: " + resultado);
    }
}

Resumo e Conclusão

Funções e procedimentos são fundamentais na programação para modularizar e organizar o código. Eles permitem a reutilização de código e facilitam a manutenção e compreensão de programas complexos.

Matrizes são amplamente usadas para representar dados tabulares ou realizar operações matemáticas, enquanto vetores são usados para armazenar sequências de elementos. Ambos desempenham um papel crucial no desenvolvimento de algoritmos eficientes.
