Introdução a JavaScript

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico em páginas web. Nesta aula, vamos cobrir os fundamentos de JavaScript e ver alguns exemplos práticos de como ele pode ser usado para manipular elementos na página.

Exemplo 1: Exibir uma mensagem de alerta

O código abaixo exibe uma mensagem de alerta na tela:

<script>
alert("Olá, bem-vindo ao JavaScript!");
</script>

Quando você carrega a página com esse código, uma caixa de alerta será exibida com a mensagem.

Exemplo 2: Manipulando o conteúdo de um elemento HTML

O código abaixo altera o texto de um parágrafo quando o botão é clicado:

<p id="meu-paragrafo">Este é o texto original.</p>
<button onclick="mudarTexto()">Clique aqui</button>

<script>
function mudarTexto() {
document.getElementById("meu-paragrafo").innerHTML = "O texto foi alterado!";
}
</script>

Ao clicar no botão, o texto do parágrafo será substituído pelo novo texto.

Exemplo 3: Somando dois números

O código abaixo permite que o usuário insira dois números e exibe a soma deles:

<input type="number" id="num1" placeholder="Digite o primeiro número">
<input type="number" id="num2" placeholder="Digite o segundo número">
<button onclick="somar()">Somar</button>

<p id="resultado"></p>

<script>
function somar() {
var n1 = parseFloat(document.getElementById("num1").value);
var n2 = parseFloat(document.getElementById("num2").value);
var resultado = n1 + n2;
document.getElementById("resultado").innerHTML = "O resultado é: " + resultado;
}
</script>

Após inserir os números e clicar em "Somar", o resultado da soma será exibido.
