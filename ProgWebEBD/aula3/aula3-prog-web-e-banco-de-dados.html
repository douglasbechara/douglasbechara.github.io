Introdução a JavaScript

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico em páginas web. Nesta aula, vamos cobrir os fundamentos de JavaScript e ver alguns exemplos práticos de como ele pode ser usado para manipular elementos na página.

Exemplo 1: Exibir uma mensagem de alerta

O código abaixo exibe uma mensagem de alerta na tela:

<script>
alert("Olá, bem-vindo ao JavaScript!");
</script>

Quando você carrega a página com esse código, uma caixa de alerta será exibida com a mensagem.

Exemplo 2: Manipulando o conteúdo de um elemento HTML

O código abaixo altera o texto de um parágrafo quando o botão é clicado:

<p id="meu-paragrafo">Este é o texto original.</p>
<button onclick="mudarTexto()">Clique aqui</button>

<script>
function mudarTexto() {
document.getElementById("meu-paragrafo").innerHTML = "O texto foi alterado!";
}
</script>

Ao clicar no botão, o texto do parágrafo será substituído pelo novo texto.

Exemplo 3: Somando dois números

O código abaixo permite que o usuário insira dois números e exibe a soma deles:

<input type="number" id="num1" placeholder="Digite o primeiro número">
<input type="number" id="num2" placeholder="Digite o segundo número">
<button onclick="somar()">Somar</button>

<p id="resultado"></p>

<script>
function somar() {
var n1 = parseFloat(document.getElementById("num1").value);
var n2 = parseFloat(document.getElementById("num2").value);
var resultado = n1 + n2;
document.getElementById("resultado").innerHTML = "O resultado é: " + resultado;
}
</script>

Após inserir os números e clicar em "Somar", o resultado da soma será exibido.

Exemplo 4: Declaração de Variáveis

Em JavaScript, podemos declarar variáveis usando var, let e const. Vamos ver como cada uma funciona.

O código abaixo demonstra como declarar variáveis:

// Declaração de uma variável que pode ser alterada var mensagem = "Olá, Mundo!"; // 'var' permite que a variável seja redeclarada e reassinada console.log(mensagem); // Exibe 'Olá, Mundo!' no console // Declaração de uma variável de escopo limitado let nome = "João"; // 'let' é usado para variáveis cujo valor pode mudar, mas não pode ser redeclarada no mesmo escopo console.log(nome); // Exibe 'João' no console // Declaração de uma constante const pi = 3.14; // 'const' é usado para valores constantes que não podem ser alterados console.log(pi); // Exibe 3.14 no console

Palavras-chave:

  • var: Declara uma variável global ou localmente na função sem restrições de escopo de bloco. Pode ser redeclarada e reassinada.
  • let: Declara uma variável com escopo de bloco, permitindo reassinamento, mas não redeclaração no mesmo escopo.
  • const: Declara uma variável com escopo de bloco que não pode ser reassinada após a inicialização.

Exemplo 5: Criando e Chamando Funções

Funções em JavaScript permitem encapsular lógica que pode ser reutilizada em várias partes do código.

O código abaixo demonstra como criar e chamar funções:

// Função que exibe uma mensagem de boas-vindas function boasVindas(nome) { // A função 'boasVindas' recebe um parâmetro 'nome' return "Bem-vindo, " + nome + "!"; // A função retorna uma saudação personalizada } console.log(boasVindas("Maria")); // Chama a função com o argumento 'Maria' e exibe 'Bem-vindo, Maria!' no console // Função que soma dois números function somar(a, b) { // A função 'somar' recebe dois parâmetros: 'a' e 'b' return a + b; // A função retorna a soma de 'a' e 'b' } console.log(somar(5, 3)); // Chama a função com os argumentos 5 e 3, e exibe '8' no console

Palavras-chave:

  • function: Define uma função em JavaScript.
  • return: Retorna um valor de uma função e encerra a execução dessa função.
  • console.log(): Exibe uma mensagem no console do navegador para fins de depuração.

Exemplo 6: Estruturas de Repetição (Loops)

Os loops permitem executar um bloco de código várias vezes de acordo com uma condição.

O código abaixo demonstra como usar um loop for:

// Loop 'for' que itera de 0 a 4 for (let i = 0; i < 5; i++) { // A variável 'i' começa em 0, e o loop continua enquanto 'i' for menor que 5 console.log("Número: " + i); // Exibe o valor de 'i' no console a cada iteração } // Loop 'while' que itera enquanto uma condição for verdadeira let j = 0; while (j < 5) { // O loop continua enquanto 'j' for menor que 5 console.log("Contagem: " + j); // Exibe o valor de 'j' no console j++; // Incrementa 'j' em 1 a cada iteração }

Palavras-chave:

  • for: Estrutura de repetição que itera um bloco de código um número específico de vezes.
  • while: Estrutura de repetição que itera um bloco de código enquanto a condição especificada for verdadeira.
  • console.log(): Exibe uma mensagem no console do navegador para fins de depuração.

Exemplo 7: Manipulando o DOM (Document Object Model)

O DOM permite que JavaScript interaja com o conteúdo de uma página HTML. Podemos usar JavaScript para alterar elementos HTML dinamicamente.

O código abaixo demonstra como alterar o conteúdo de um elemento HTML usando JavaScript:

// Alterando o conteúdo de um elemento HTML document.getElementById("demo").innerHTML = "Texto alterado!"; // Encontra o elemento com o id 'demo' e altera seu conteúdo

Palavras-chave:

  • document: Representa o documento HTML carregado no navegador.
  • getElementById(): Método que retorna o elemento com o id especificado.
  • innerHTML: Propriedade que define ou retorna o conteúdo HTML de um elemento.
